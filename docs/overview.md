# Descripción General

## Descripción

La plataforma es una solución de atención automatizada al cliente que integra WhatsApp Cloud API, Chatwoot, n8n, OpenAI, PostgreSQL y Redis para gestionar conversaciones, consultar información empresarial y proporcionar respuestas inteligentes mediante canales de mensajería.

La plataforma permite recibir mensajes de clientes, procesar solicitudes mediante flujos automatizados, generar respuestas contextuales utilizando inteligencia artificial y enviar respuestas nuevamente a través de WhatsApp.

## Características Principales

- Atención automatizada al cliente.
- Integración con WhatsApp Cloud API.
- Respuestas conversacionales impulsadas por Inteligencia Artificial.
- Orquestación de procesos mediante n8n.
- Gestión de contexto utilizando Redis.
- Acceso a información estructurada mediante PostgreSQL.
- Despliegue basado en Docker.
- Arquitectura escalable para entornos empresariales.

## Objetivos del Proyecto

- Reducir tiempos de respuesta.
- Automatizar consultas repetitivas.
- Mejorar la experiencia del cliente.
- Centralizar procesos de comunicación.
- Proporcionar una arquitectura reutilizable para automatización de atención al cliente.
